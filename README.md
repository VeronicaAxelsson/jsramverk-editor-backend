# Texteditor

REST API made in the course JSramverk.

# Get started with developing

-   Download repo
-   Install requirements:
    ```text
    npm install
    ```
-   .env

    -   ATLAS_USERNAME
    -   ATLAS_PASSWORD
    -   PORT (default 1337)

-   Run development server:
    ```text
    npm run start
    ```
-   To format code using prettier:
    ```text
      npm run format
    ```
